## 📌 Problem Statement

This project involves performing file handling operations on a given file **CS4.txt**.

### Tasks:

1. Read file using:

   * read()
   * readline()
   * readlines()
2. Print:

   * Total number of lines
   * First 2 lines
   * Last 2 lines
3. Count keywords:

   * INFO
   * WARNING
   * ERROR
4. Store counts in a dictionary
5. Create filtered files:

   * info_logs.txt
   * warning_logs.txt
   * error_logs.txt
6. Implement search feature:

   * Take user input
   * Display matching lines
   * Save results in search_result.txt
7. Perform file pointer operations using seek()

---

## ⚙️ Approach

* Opened the file in read mode
* Used different file reading methods (read, readline, readlines)
* Counted log types using a dictionary
* Filtered lines into separate files based on keywords
* Implemented user-based keyword search
* Used seek() to move file pointer to different positions
